# Simple Portfolio Website

## Project Description
This is a basic portfolio webpage created using HTML with internal CSS.  
It contains sections like About Me, Skills, Projects, and Contact.  
The purpose of this project is to practice using Visual Studio Code, Git, and GitHub for software development.

---

## How to Run the Project
1. Download or clone the repository.
2. Open the folder in Visual Studio Code.
3. Open the file `Assignment4.html` in a web browser (Chrome, Edge, Firefox).

---

## Features
- Simple header with name and details
- About section
- Skills list
- Small project list
- Contact information
